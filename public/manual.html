スタッフ向け操作ガイド

注文・在庫管理を行う医院スタッフの方向け

1. ログイン

画面：ログイン
  1. ブラウザで dental-app-brown.vercel.app を開く
  2. 管理者から渡された パスワード を入力する
  3. ログイン をタップする
💡 ブックマーク登録を推奨
スマートフォンのブラウザでホーム画面に追加すると、アプリのように使えます。
⚠ 管理者ログインについて
ログイン画面下部の「管理者ログイン」はシステム管理者専用です。スタッフは操作不要です。

2. メニュー画面

画面：メニュー

ログイン後に表示されるホーム画面です。以下の機能にアクセスできます。

ボタン内容
🛒 注文医療材料・消耗品の注文
📋 注文履歴過去の注文確認・再注文・キャンセル申請
📦 在庫管理院内在庫の使用・補充の記録

3. 注文画面

画面：注文

3-1. 商品を探す・カートに追加

  1. 画面上部の検索欄に商品名・コード・メーカー名を入力する
  2. カテゴリボタン（画面上部またはPC左サイドバー）で絞り込むこともできる
  3. 商品カードの ＋ カートに追加 をタップする
  4. カートに商品が入ると画面右下に 🛒 カートボタン が表示される

3-2. バーコードスキャン

  1. 📷 バーコードでカートに追加 をタップ
  2. カメラが起動するので商品のバーコードに向ける
  3. 読み取りに成功すると自動でカートに追加される
ℹ バーコードが読み取れない場合
商品マスタにバーコードが未登録の可能性があります。管理者に登録を依頼してください。

3-3. お気に入り

よく注文する商品は❤️ボタンでお気に入り登録できます。登録した商品は画面上部の「❤️ お気に入り」に表示されます。

  • 商品カード右上の 🤍 をタップ → ❤️に変わりお気に入り登録
  • もう一度タップで解除

3-4. 注文を確定する

  1. 🛒カートボタンをタップしてカートを開く
  2. 数量を確認・修正する（「−」「＋」ボタンまたは数字を直接入力）
  3. 注文確認へ → をタップ
  4. 注文者名（担当者名）を入力する　※必須
  5. 内容を確認して ✓ 注文を確定する をタップ
  6. 「注文が完了しました」画面が表示されたら完了
⚠ 注文者名は必須項目です
注文担当者名を入力しないと注文できません。フルネームで入力してください。

4. 注文履歴

画面：注文履歴

4-1. 履歴を確認する

過去の注文が月ごとにまとめて表示されます。

  • 月のヘッダーをタップすると折りたたみ・展開できる
  • 月ごとの 注文件数・合計金額 がヘッダーに表示される
  • 各注文には 進捗バー で現在のステータスが確認できる
注文受付
→
処理中
→
発送済み
→
納品済み

「明細 ▼」ボタンで注文内容の詳細（商品・数量・金額）を表示できます。

検索・絞り込み：

  • 上部の検索欄で納品書番号・商品名・担当者名を検索
  • ステータスのプルダウンで絞り込み
  • 日付範囲で期間指定

4-2. 再注文

  1. 再注文したい注文カードの 再注文 をタップ
  2. 数量確認画面が開く。数量を変更する場合は「−」「＋」で調整
  3. この内容で再注文する をタップ
💡 再注文のヒント
数量を0にした商品は注文に含まれません。不要な商品は0にしてから確定できます。

4-3. キャンセル申請

ステータスが「注文受付」の注文のみキャンセル申請できます。

  1. キャンセルしたい注文の キャンセル をタップ
  2. 確認ダイアログで「OK」をタップ
  3. ステータスが キャンセル申請中 に変わる
  4. 管理者が確認後、正式にキャンセル処理される
⚠ 注意
「処理中」以降のステータスはキャンセルできません。急ぎの場合は管理者に直接連絡してください。

5. 在庫管理

画面：在庫管理

院内の消耗品・材料の在庫数を管理します。画面下部の「記録」「履歴」タブで切り替えます。

5-1. 在庫を記録する（使用 / 補充）

クイック操作（±1）：

  1. 商品カードを探す（検索欄・置き場所フィルターで絞り込み可）
  2. 使用 -1 または 補充 +1 をタップ
  3. 画面上部に「✓ 記録しました」のメッセージが表示されれば完了

数量を指定して記録（···ボタン）：

  1. 商品カードの ··· ボタンをタップ
  2. 底部から操作パネルが開く
  3. 「使用」または「補充」を選択する
  4. 「−」「＋」ボタンまたは数字入力で数量を指定
  5. 記録する をタップ
💡 置き場所フィルター
処置室・倉庫など置き場所ごとに絞り込んで表示できます。管理者が設定した置き場所のボタンが検索欄の下に表示されます。

5-2. バーコードスキャンで記録

  1. 📷 スキャンして記録 をタップ
  2. 商品のバーコードを読み取る
  3. 操作パネルが自動で開くので「使用」/「補充」と数量を選んで記録

5-3. 在庫数を直接修正（棚卸調整）

実際の在庫数と記録が合わない場合に使います。

  1. 修正したい商品カードの 在庫数（大きい数字）をタップ
  2. 入力欄に正しい在庫数を入力
  3. ✓ ボタンまたはEnterキーで確定（✕で取消）
  4. 「棚卸調整」として履歴に記録される

5-4. 出し入れ履歴を見る

  1. 画面下部の 「履歴」タブ をタップ
  2. 「今日」「今週」「すべて」で期間を絞り込める
  3. スタッフ名のボタンで担当者別に絞り込める

履歴の種類：

種類意味表示色
使用在庫を使った赤系
補充在庫を補充した緑系
棚卸調整直接数量を修正したオレンジ系
⚠ 発注が必要な商品
在庫数が最低在庫数以下になると 発注必要 バッジが表示されます。「注文画面へ →」ボタンから注文画面に移動して発注してください。

管理者向け操作ガイド

管理画面を操作するシステム管理者向け

6. 管理者ログイン

  1. ログイン画面下部の「管理者ログイン」をタップ
  2. 登録済みの メールアドレス と パスワード を入力
  3. ログインすると管理画面に移動する
⚠ セキュリティ
管理者アカウントのパスワードは他の人と共有しないでください。スタッフには専用のスタッフパスワードを発行してください。

7. 注文管理

管理画面：注文

医院スタッフからの注文を管理します。

注文一覧の確認

  • 医院名・ステータス・日付で絞り込み検索ができる
  • 注文番号（DN-）をクリックすると注文詳細を確認できる
  • 注文受付 → ステータスを更新して業務フローを進める

ステータスの流れ

注文受付
→
処理中
→
発送済み
→
納品済み

キャンセル申請への対応

医院からキャンセル申請が来た注文は キャンセル申請中 と表示されます。内容を確認してステータスを「キャンセル」に変更してください。

8. 発注管理

管理画面：発注

仕入先への発注を管理します。

  • 発注：発注書の作成・管理
  • 🛒 プール：発注候補商品のまとめ置き場。複数注文から必要商品をプールに集めてまとめて発注できる
  • 推奨発注：在庫が少ない商品を自動で抽出してリスト化

発注書の作成手順

  1. 「発注」→「新規作成」をクリック
  2. 仕入先を選択し、商品・数量を入力
  3. 保存すると発注番号が発行される
  4. 仕入先への発注後、ステータスを「発注済み」に更新

9. 仕入納品

管理画面：仕入納品

仕入先から商品が届いた際に入荷を記録します。

  1. 「仕入納品」を開く
  2. 対象の発注書を選択して「入荷登録」をクリック
  3. 実際に届いた数量を確認・入力する
  4. 登録すると在庫が自動で増加する

10. 医院納品

管理画面：医院納品

医院への商品納品を記録します。

  • 注文をもとに納品書を作成できる
  • 「納品書印刷」で印刷用フォーマットを開ける
  • 納品完了後にステータスを「納品済み」に更新する（医院の注文履歴に反映される）

11. 請求管理

管理画面：請求

請求書の作成

  1. 「請求」→「新規作成」をクリック
  2. 対象の医院・請求月を選択
  3. 該当期間の納品データが自動で読み込まれる
  4. 内容を確認して「保存」をクリック

一括請求

「一括請求」機能で複数医院分の請求書を一度に作成できます。

請求書の印刷

  1. 請求書一覧から対象を開く
  2. 「印刷」ボタンをクリック
  3. ブラウザの印刷ダイアログが開くのでPDF保存または印刷

売掛金・入金管理

  • 売掛金台帳：未回収金額の一覧
  • 銀行CSV消込：銀行の入金データをCSVで取り込んで自動照合

12. マスター管理

管理画面：マスター

各種マスターデータの管理を行います。「マスター」メニューから各設定ページに移動できます。

12-1. 商品マスタ

  • 取り扱い商品の登録・編集
  • 商品名・コード・メーカー・仕入値・定価・カテゴリを管理
  • 「有効」フラグをOFFにすると注文画面に表示されなくなる
  • 商品画像（別ページ）：楽天APIで商品画像を一括取得できる

12-2. 得意先（医院）マスタ

  • 取引医院の情報を管理（医院名・住所・締日など）
  • 請求書・納品書に反映される

12-3. ユーザー管理

医院スタッフのログインパスワードを設定します。

  1. 「マスター」→「ユーザー」を開く
  2. パスワードを設定したいユーザーの入力欄に新しいパスワードを入力（6文字以上）
  3. 設定する をクリック
  4. ✓ 設定済み と表示されたら完了
💡 パスワードの共有方法
設定したパスワードは画面上に表示されます。スタッフへの共有は口頭またはメモで行ってください。

12-4. お知らせ管理

医院スタッフの注文画面に表示されるお知らせを管理します。

  1. 「管理画面」上部ナビの「📢 お知らせ」を開く
  2. タイトル（必須）と本文（任意）を入力して「投稿する」をクリック
  3. 投稿したお知らせは有効/無効の切り替えと削除ができる
ℹ 表示される場所
有効なお知らせは医院スタッフの注文画面の上部に緑のボーダー付きで表示されます。スタッフが✕を押すと非表示になります（再ログインすると再表示）。

12-5. 医院在庫アイテム

医院スタッフの在庫管理画面に表示される商品の 置き場所 と 最低在庫数 を設定します。

  1. 「マスター」→「医院在庫」を開く
  2. 商品の「置き場所」列をクリックして場所を入力（例：処置室、倉庫、棚A）
  3. 「最低在庫」列に数値を入力（この数以下になると発注必要バッジが表示される）
  4. フォーカスを外すと自動保存される（Enterキーでも保存）
💡 置き場所の活用
置き場所を設定すると、スタッフの在庫画面で場所ごとにグループ表示・絞り込みができるようになります。「処置室」「受付」「倉庫」などの単位で管理することをおすすめします。
